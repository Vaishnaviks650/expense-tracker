Expense Tracker 💰

A simple and responsive Expense Tracker web application built using HTML, CSS, and JavaScript. This application helps users manage their income and expenses, track their balance, and visualize spending habits in a clean and user-friendly interface.

🚀 Features
Add income and expense transactions
Automatically calculate total balance
Display total income and total expenses
Delete transactions
Real-time balance updates
Responsive design for mobile and desktop devices
Local storage support to save data between sessions

⚙️ How It Works
Enter a transaction description.
Enter the amount:
Positive value → Income
Negative value → Expense
Click Add Transaction.
The application updates:
Current Balance
Total Income
Total Expenses
Transactions are stored in the browser using Local Storage.
